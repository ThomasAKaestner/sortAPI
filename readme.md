# Sort API 
This project is just a small api which takes as a request an array/tupel.
It can sort the array with 4 different algorithms.

## How to call the API
* Post Request
* /sort/"your sort algo comes in here"

## Example Request


### Bubble Sort 
* api: /sort/bubble
### Quick Sort
* api  /sort/quick
### Merge Sort 
* api  /sort/merge
### Bogo Sort 
* api  /sort/bogo
