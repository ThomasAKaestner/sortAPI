# Sort API 
This project is just a small api. The request takes an array. After sorting the array it returns it with comparisons(int) made.

## How to call the API
* Post Request
* /sort/"your sort algo comes in here"

## Example Request


### Bubble Sort 
* api: /sort/bubble
### Quick Sort
* api  /sort/quick
### Merge Sort 
* api  /sort/merge
### Bogo Sort 
* api  /sort/bogo
